# traincalc-met-dataset
Compendium-style MET (Metabolic Equivalent of Task) dataset used by TrainCalc. Includes CSV/JSON exports, schema, sources, and usage notes.

TrainCalc MET Dataset
A compendium-style MET dataset (Metabolic Equivalent of Task) curated for calorie estimation tools.
	•	Live table: https://traincalc.com/met-values
	•	Compare tool: https://traincalc.com/met-compare
	•	Formats: data/met.csv, data/met.json
	•	Formula used in tools: Calories = MET × weight(kg) × hours

About the data
This repository contains a curated, tool-friendly dataset mapped to compendium-style activity descriptions (pace, incline, sport, etc.). Sources and attribution are listed below.
